gCellV0
=======

Calculation algorithmic OrGraphs

gCellV0 is based in part on the work of
the Qwt project (http://qwt.sf.net).

Compil:

win32:
- cd {src_dir_path}\gcellv0
- qmake
- mingw32-make

required libraries:
- qwt plot 6.1.0 (http://qwt.sf.net)
